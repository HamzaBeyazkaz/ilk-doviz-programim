# 🚀 Canlı Döviz Takip ve Hesaplama Uygulaması
Bu uygulama, Python ve `tkinter` kütüphanesi kullanılarak geliştirilmiştir. 

### Özellikler:
- API üzerinden anlık dolar kurunu çeker.
- Kullanıcıdan alınan miktarı TL'ye çevirir.
- Kurun durumuna göre "Pahalı/Ucuz" analizi yapar.
- Bağımsız bir `.exe` dosyası olarak çalışabilir.
